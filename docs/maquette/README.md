## Design System

- **Style** : Glassmorphism avec `backdrop-filter: blur(12px)`
- **Palette** : Material Design 3 (primary: #6750a4, emerald-ai: #10b981, electric-indigo: #6366f1)
- **Typographie** : Inter (body), JetBrains Mono (données)
